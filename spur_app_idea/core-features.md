# Core Features: White-Label Food Ordering App

## 1. User Authentication
- Sign up, login, password reset
- User profile management

## 2. Language & Accessibility
- **Primary language selection during signup**
- **Language recommendation to restaurant staff** (when user selects their language, restaurant gets notified that waiter should speak that language)
- Multi-language support for app interface

## 3. Restaurant Branding
- Custom logo upload and display
- Custom color scheme (primary, secondary colors)
- Restaurant name and contact information
- Social media links
- Restaurant description and story

## 4. Menu Management
- Menu categories (e.g., Starters, Mains, Drinks, Desserts)
- Menu items with:
  - Name and description
  - Price
  - Image
  - Dietary information (vegetarian, vegan, gluten-free, etc.)
  - Allergen warnings
  - Customization options (add-ons, toppings, etc.)
  - Availability status (in stock, out of stock)

## 5. Cart & Checkout
- Add/remove items from cart
- Modify item quantities
- View cart with total price
- Apply discount codes/promotions
- Choose delivery or pickup
- Select delivery address
- Choose payment method
- Place order

## 6. Order Management
- **Order Status Tracking:**
  - Order received
  - Preparing
  - Ready for pickup/delivery
  - Out for delivery
  - Completed
  - Cancelled
- Order history for users
- Real-time order updates
- Order notifications

## 7. Notifications
- Order status updates
- Promotional notifications
- Special offers
- New menu items
- Restaurant announcements

## 8. Admin Panel
- **Restaurant Management:**
  - Update restaurant info (name, logo, colors, contact)
  - Manage menu (add/edit/delete categories and items)
  - Set opening hours
  - Configure delivery zones and fees
- **Order Management:**
  - View current orders
  - Update order status
  - View order history
  - Manage customer feedback
- **Analytics:**
  - Sales reports
  - Popular items
  - Customer insights

## 9. Multi-Tenancy (White-Label Support)
- Support multiple restaurants on same platform
- Each restaurant has independent:
  - Branding (logo, colors, name)
  - Menu
  - Orders
  - Admin users
  - Settings

## 10. App Settings
- Opening hours configuration
- Delivery and pickup options
- Payment methods (cash, card, online)
- Delivery radius and fees
- Minimum order amounts
- Tax rates

## 11. Customer Features
- User reviews and ratings
- Favorite items
- Reorder from history
- Multiple delivery addresses
- Guest checkout option

## 12. Restaurant Staff Features
- **Language notifications** (when customer orders in specific language)
- Order queue management
- Kitchen display system
- Staff login and roles
