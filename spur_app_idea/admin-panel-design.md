# Admin Panel Design: White-Label Food Ordering App

## Admin Panel Sections

### 1. **Dashboard**
- Overview of today's orders
- Revenue summary
- Popular items
- Quick actions (mark as open/closed, etc.)

### 2. **Restaurant Settings**
- **Branding:**
  - Upload logo (with disclaimer about app icon limitation)
  - Set primary/secondary colors
  - Restaurant name and contact info
  - Social media links
- **Business Settings:**
  - Opening hours
  - Delivery zones and fees
  - Payment methods
  - Minimum order amounts

### 3. **Menu Management**
- **Categories:**
  - Add/edit/delete menu categories
  - Reorder categories
- **Menu Items:**
  - Add new items with name, description, price, image
  - Edit existing items
  - Set availability (in stock/out of stock)
  - Add customization options (toppings, add-ons)
  - Set dietary info (vegetarian, gluten-free, etc.)
  - **Allergy Alerts:**
    - Mark items containing common allergens (nuts, dairy, shellfish, etc.)
    - Add custom allergy warnings for specific items
    - Set allergen icons/indicators for menu display

### 4. **Order Management**
- **Current Orders:**
  - View all active orders
  - Update order status (received → preparing → ready → completed)
  - Add notes to orders
  - **Allergy Alert System:**
    - Highlight orders with allergy concerns
    - Notify kitchen staff of allergy requirements
    - Confirm allergy-safe preparation
- **Order History:**
  - Search and filter past orders
  - Export order data

### 5. **Staff Management**
- **User Roles:**
  - Owner (full access)
  - Manager (can manage menu and orders)
  - Staff (can only view and update orders)
- **Language Notifications:**
  - View when customers order in specific languages
  - Assign staff to handle specific language orders

### 6. **Analytics**
- Sales reports
- Popular items
- Customer insights
- Revenue trends

### 7. **Customer Management**
- View customer profiles
- Order history per customer
- Customer feedback and reviews
- **Allergy Profiles:**
  - Track customer allergy preferences
  - Flag customers with severe allergies
  - Provide allergy-safe recommendations

## User Roles & Permissions

### 1. **Owner**
- Full access to everything
- Can manage billing and subscriptions
- Can add/remove staff members

### 2. **Manager**
- Can manage menu, orders, staff
- Cannot access billing/analytics
- Can update restaurant settings

### 3. **Staff**
- Can only view and update orders
- Can see allergy alerts and language preferences
- Cannot modify menu or settings

## Allergy Alert Features

### For Customers:
- Clear allergen indicators on menu items
- Warning popups when ordering items with allergens
- Ability to mark personal allergies in profile
- Allergy-safe menu filtering options

### For Restaurant Staff:
- Prominent allergy alerts on order tickets
- Kitchen display system highlights allergy orders
- Confirmation system for allergy-safe preparation
- Training materials for handling allergy orders
